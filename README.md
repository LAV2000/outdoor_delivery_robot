# outdoor_delivery_robot

Outdoor Delivery Robot (HUST)
## Yêu cầu
- Ubuntu [20.04](https://releases.ubuntu.com/focal/)
- ROS2 [Foxy](https://docs.ros.org/en/foxy/Installation.html)

|--|--|

|odr_control|Cấu hình bộ điều khiển cho Robot|
